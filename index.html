What I Do

Vanilla JavaScript

Vanilla JavaScript refers to the use of pure, plain JavaScript without any additional libraries or frameworks. It allows developers to work directly with the native capabilities of web browsers to create interactive and dynamic web applications.

HTML

HTML (Hypertext Markup Language) and CSS (Cascading Style Sheets) are fundamental technologies for web development. HTML is used to structure the content of a web page, defining elements like headings, paragraphs, lists, and links.

CSS

CSS is used for styling and layout, allowing developers to control the appearance of HTML elements by defining properties like colors, fonts, and positioning.

React

React is an open-source JavaScript library developed by Facebook that is widely used for building user interfaces (UIs) for web applications. It was first released in 2013 and has since gained immense popularity in the web development community.
